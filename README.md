Sistema multi-agente para investigación automatizada usando LangChain y Ollama.

Requisitos Previos
1. Instalar Ollama
    Descarga e instala Ollama desde ollama.ai

2. Descargar los modelos necesarios
    bashollama pull qwen2.5
    ollama pull llama3.2:3b

3. Instalar dependencias de Python
    bashpip install -r requirements.txt

Cómo Ejecutar
    python .\main.py

Los archivos __pycache__ están deshabilitados por defecto
